## About
car-rental is a simple car rental webapp.

## Technology stack
**Java: 17**

**Build:** Maven

**Core:** Spring MVC, Spring Security, Spring Data JPA, Hibernate, Spring Test, MySql/H2

**Additional:** Lombok, Hibernate Validator, MapStruct, Mockito

## How to build
Required: Tomcat, MySql
```sh
mvn clean install
```
